# 🌟 Sanjay's Personal Portfolio Website

Welcome to my personal portfolio — a clean, modern, and visually immersive landing page designed to introduce myself, showcase my works, and make it easy for visitors to connect with me.

---

## 🔧 Technologies Used
- **HTML5**
- **CSS3**
- **JavaScript**

---

## 🎨 Design Features
- Full-screen background with cinematic visuals
- Smooth and minimal UI with icon-based navigation
- Sections for **Welcome**, **About (Myself)**, **Projects (My Works)**, and **Contact**
- Mobile-friendly layout with consistent dark/light theme aesthetics

---

## 💡 Purpose
This project serves as a **digital handshake** — a place where I welcome potential collaborators, recruiters, and like-minded creators.  
It’s designed to reflect **personality**, **creativity**, and **ease of access**.

---

## ✅ Features
- Responsive layout for both desktop and mobile
- Quick navigation with hover effects
- Personalized intro message
- Copyright footer with dynamic year
